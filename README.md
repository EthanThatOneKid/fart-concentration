# Fart Concentration

Test your memory using fart sounds!

## Play the game

Play online: <https://concentration.fart.tools>

Concentration is a card game in which all of the cards are laid face down on a surface and two cards are flipped face up over each turn. The object of the game is to turn over pairs of matching cards. The game ends when the last pair has been picked up.
